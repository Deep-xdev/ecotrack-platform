# EcoTrack Platform

A frontend dashboard designed to visualize environmental and waste-related data in a clear and structured way.

## Features
- Responsive UI using HTML and CSS
- Clear visual hierarchy for data sections
- Focus on readability and layout clarity
- Deployed using Netlify

## Tech Stack
- HTML5
- CSS3
- JavaScript

## What I Learned
- Structuring UI for better user experience
- Building responsive layouts
- Organizing frontend code cleanly
